Comment mettre une musique dans un qrcode ?

Ouvrir la fenêtre de saisie de la musique
Cliquer sur le bouton
Une boite de dialogue avec une zone de saisie va s'ouvrir
Copier le lien téléchargeable de la musique
Google Drive
  • Aller sur internet, se rendre sur son compte Google Drive (https://drive.google.com), se connecter éventuellement
  • Faire clic-droit sur le fichier audio en question
  • Cliquer sur "Obtenir le lien partageable"
  • Revenir sur l'application QRLudo
  • Vidéo résumant les étapes ci dessus
Dropbox
  • Aller sur internet, se rendre sur son compte Dropbox (https://www.dropbox.com), se connecter éventuellement
  • Survoller avec la souris le fichier audio en question
  • Cliquer sur "Partager", un popup va s'ouvrir
  • En bas à droite du popup ouvert, cliquer sur "Créer un lien"
  • Toujours sur le popup, cliquer sur "Copier le lien" en bas à droite
  • Revenir sur l'application QRLudo
  • Vidéo résumant les étapes ci dessus
Coller le lien récemment copié
  • Faire clic-droit sur la zone de saisie musique
  • cliquer sur "Coller le lien" (ou avec le clavier faire la combinaison de touches Ctrl + V)
Ajouter une question au projet
  • Saisissez le texte de votre question dans le champ puis cliquez sur le bouton
  • Une boîte de dialogue apparaît : elle contient votre question et vous permet d'ajouter de nouvelles réponses à votre question (voir Ajouter une réponse à une question/En modifiant directement la réponse).
  • Un nouveau champ apparaît : il contient toutes les informations de la question que vous venez de créer.
Ajouter une reponse à une question
En modifiant directement la question
  • Cliquez sur le bouton de la question que vous voulez modifier.
  • Une boîte de dialogue qui contient les informations de la question apparaît.
  • Dans le champ , saisissez le texte de votre réponse puis cliquez sur le bouton pour ajouter votre réponse.
  • Votre réponse apparaît et est ajoutée par défaut.
  • Enfin, cliquez sur le bouton pour valider vos modifications.

  •   Avec le champ , vous pouvez modifier le message qui sera lu par la synthèse vocale de QRLudo lorsque cette réponse sera scannée. Par défault, le message est celui indiqué dans les paramètres.

  • Votre réponse apparaît dans la description de la question.
En créant une réponse et en l'ajoutant à la question
  • Cliquez sur .
  • Dans le champ , saisissez votre réponse puis validez la en cliquant sur le bouton
  • Votre réponse apparaît.
  • Cliquez sur pour retourner à la page des questions.
  • Cliquez sur le bouton de la question que vous voulez modifier.
  • Une boîte de dialogue apparaît.
  • Cochez la case correspondant à la réponse que vous souhaitez ajouter puis cliquez sur pour enregister vos modifications.
Retirer une réponse à une question
  • Cliquez sur le bouton qui correspond à question que vous souhaitez modifier.
  • Décochez la case correspondant à la réponse que vous voulez retirer.
  • Cliquez sur pour enregister vos modifications.
Modifier une question
  • Cliquez sur le bouton qui correspond à la question que vous souhaitez modifier.
  • Modifiez l'intitulé de votre question dans le champ .
  • Cliquez sur pour enregister vos modifications.
Supprimer une réponse du projet
  • Cliquez sur .
  • Cliquez sur le bouton de la réponse que vous souhaitez supprimer.
Enregistrer un projet
  • Vérifiez que vous avez entré le nom de votre projet.
  • Cliquez sur
  • Une boîte de dialogue apparaît : elle vous permet de sélectionner l'endroit où votre projet va être sauvegarder.
  • Un message vous indique que votre projet a bien été enregister.
Importer un projet
  • Cliquez sur
  • Sélectionnez le nom du dossier qui contient votre projet.
  • Tous les éléments de votre projet sont affichés.
